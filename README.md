# ToDoList React

Веб-приложения для написания списка задач.

### Функционал: 
- Добавление задачи; 
- Отметка о выполнения задачи;
- Удаление задачи;
- Удаление выполненных задач;
- Удаление всех задач.


### Стек
- React;
- HTML5;
- CSS3;
- CSS Modules;
- JavaScript.
